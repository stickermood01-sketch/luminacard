# **App Name**: LuminaCard

## Core Features:

- AI Flashcard Generation Tool: Analyze uploaded PDFs and use AI as a tool to extract key concepts and automatically create high-quality flashcards.
- Intelligent Spaced Repetition: An algorithmic study schedule based on Supabase-stored performance metrics to optimize long-term memory retention.
- Study Progress Dashboard: Visual overview of learning metrics, upcoming reviews, and mastery levels using clean Notion-inspired charts.
- Secure PDF Repository: Cloud-based storage via Supabase to manage and organize original source documents for card referencing.
- Anki-Inspired Review Interface: A minimal, distraction-free flashcard interface with rapid keyboard and touch gestures for quick grading.
- Universal Authentication: Integrated Supabase Auth for seamless cross-device synchronization and secure study progress.

## Style Guidelines:

- The palette follows a midnight productivity theme. Primary: Vibrant Indigo (#6366F1). Background: Deep Matte Slate (#0B0D14) for high-contrast focus. Accent: Clear Cerulean (#38BDF8) to highlight active UI elements and progress bars.
- Recommend 'Inter', a grotesque-style sans-serif for its machined, objective look and exceptional readability on screens of all sizes, ensuring a Notion-like minimalist aesthetic.
- Minimalist grid layout featuring narrow content widths to reduce horizontal scanning, consistent with modern workspace tools and highly responsive for mobile studying.
- Lean, thin-stroke monolinear icons for actions, reinforcing a sense of precision and technical sophistication.
- Swift, crisp transitions between card states and page views to minimize friction during high-frequency study sessions.
